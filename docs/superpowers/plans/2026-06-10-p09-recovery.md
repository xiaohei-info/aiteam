# P09 Office Recovery Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 P09 办公室页补齐首屏加载失败后的自动恢复能力，确保断线恢复后页面能自动重新拿到最新状态，而不是只能手工刷新。

**Architecture:** 保持现有 `/api/team/office/scene` 与 `/api/team/office/feed` 合同不变，只增强 `office.js` 的 polling/recovery 行为。测试先行，用 Node VM lifecycle 证明“首屏失败时未启动恢复轮询”的缺口，再以最小实现修复。

**Tech Stack:** Python pytest, Node VM-based frontend module tests, vanilla JavaScript, existing Team Panel office APIs
